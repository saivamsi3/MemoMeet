# User Guide

## Getting Started
1. Register an account
2. Add participants
3. Create meetings with notes
4. Let AI extract memories
5. Use Memo Chat to recall information

## Features
- **Participants:** Manage contacts with detailed profiles
- **Meetings:** Create and manage meeting notes
- **Memories:** AI extracts facts, concerns, goals, and more
- **Action Items:** Track commitments and follow-ups
- **Memo Chat:** Ask questions about your data
- **Analytics:** Visual insights about relationships
- **Reports:** Generate preparation reports
- **Timeline:** View relationship history
